**WSML** (**W**eb **S**erver **M**anager/**L**auncher) is a network [Python](https://www.python.org) tool for creating a local server on your computer. The tool features a range of customization options, making it easy to use for all users, from beginners to experts.

## Features

- Create a web server
- Select the port of your choice between 0 and 65535
- Select a root folder to put files on the server
- Real-time server latency analysis (update coming soon)
- Base color can be changed : Blue, Black, Green, Red, Cyan, Magenta and Yellow

## Instalation

Lastest version : `1.0`

- Install Python lastest version [here](https://www.python.org/downloads/)
- Download the latest version in [release](https://github.com/Math9898/WSML/releases/)
- Extract zip file
- Install `requests` module with `pip` in your terminal
 
  ```
  pip install requests
  ```
- Now launch the script with this command (to be adapted to the access path) :
 
  ```
  cd "c:/Users/User/Downloads/WSML/" && C:/Users/User/AppData/Local/Microsoft/WindowsApps/python3.11.exe "main.py"
  ```

## Usage

WSML's consolidated interface is very intuitive. To launch a web server, simply select option 1. If you wish to modify the port or root folder path, select option 2 and then make the modifications of your choice. To change the default color, select option 3 and choose the color of your choice. More options will be added in future updates.
